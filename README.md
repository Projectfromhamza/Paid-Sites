# Forge & Timber

Premium dual-trade services website: welding + woodwork.

## Run

```bash
npm install
npm run dev
```

## Images

Place numbered assets in `public/images/` using the filenames from the implementation plan (`01-hero.jpg` … `30-carousel-4.jpg`). Until then, the UI shows crafted fallbacks automatically.

## Customize

Edit contact details and copy defaults in `src/data/site.ts`.
